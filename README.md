# Kumpulan Doa Ramadhan 2026 🌙

Aplikasi web interaktif untuk kumpulan doa-doa Ramadhan dengan desain yang menarik dan responsif.

## Fitur Utama

✨ **Fitur-Fitur:**
- 🤲 4 doa Ramadhan (Doa Sahur, Doa Buka Puasa, Lailatul Qadr, dan Taubat)
- 📋 Sistem tab untuk filter doa berdasarkan kategori
- 📋 Tombol "Salin" untuk menyalin doa ke clipboard
- 🎨 Desain modern dengan gradien warna Ramadhan
- 📱 Responsif dan mobile-friendly
- ⚡ Animasi halus dan interaktif
- 💬 Toast notification untuk konfirmasi

## Cara Menjalankan

### Metode 1: Membuka File Langsung (Termurah & Termudah)
1. Buka file `index.html` dengan browser (Chrome, Firefox, Safari, Edge, atau browser lainnya)
2. Klik dua kali pada `index.html` atau klik kanan → "Open with" → pilih browser Anda
3. Website akan langsung muncul di browser

### Metode 2: Menggunakan Live Server (VS Code)
Jika menggunakan VS Code:
1. Install ekstension "Live Server" (oleh Ritwick Dey)
2. Klik kanan pada `index.html`
3. Pilih "Open with Live Server"
4. Website akan otomatis membuka di browser

### Metode 3: Menggunakan Python (Jika Sudah Terinstall)
```bash
# Untuk Python 3.x
python -m http.server 8000

# Kemudian buka browser dan akses:
http://localhost:8000
```

### Metode 4: Menggunakan Node.js & http-server
```bash
# Install http-server (jika belum)
npm install -g http-server

# Jalankan server di folder project
http-server

# Akses di browser
http://localhost:8080
```

## Navigasi Website

### Tab Kategori
- **Semua Doa**: Menampilkan semua doa
- **Doa Sahur**: Filter hanya doa Sahur dan Lailatul Qadr
- **Doa Buka**: Filter hanya doa Buka Puasa dan Taubat

### Interaksi
1. Klik tab kategori untuk menyaring doa
2. Klik tombol "Salin Doa" untuk menyalin ke clipboard
3. Toast notification akan muncul untuk konfirmasi
4. Hover di atas kartu untuk melihat efek animasi

## Doa yang Tersedia

### 1. Doa Sahur 🌅
**Arab:** نَوَيْتُ صَوْمَ غَدٍ عَنْ أَدَاءِ فَرْضِ شَهْرِ رَمَضَانَ هَذِهِ السَّنَةِ لِلّٰهِ تَعَالَى
**Indonesia:** Aku berniat berpuasa esok hari untuk menunaikan kewajiban bulan Ramadhan tahun ini karena Allah Ta'ala.

### 2. Doa Buka Puasa 🍽️
**Arab:** اللَّهُمَّ لَكَ صُمْتُ وَبِكَ آمَنْتُ وَعَلَيْكَ تَوَكَّلْتُ وَعَلَى رِزْقِكَ أَفْطَرْتُ
**Indonesia:** Ya Allah, untuk-Mu aku berpuasa, kepada-Mu aku beriman, kepada-Mu aku bertawakal, dan dengan rezeki-Mu aku berbuka puasa.

### 3. Doa Lailatul Qadr ✨
**Arab:** اللَّهُمَّ إِنَّكَ عَفُوٌّ تُحِبُّ الْعَفْوَ فَاعْفُ عَنِّي
**Indonesia:** Ya Allah, sesungguhnya Engkau Maha Pemaaf, menyukai pemaafan, maka maafkanlah aku.

### 4. Doa Taubat 🙏
**Arab:** أَسْتَغْفِرُ اللَّهَ الَّذِي لَا إِلَٰهَ إِلَّا هُوَ الْحَيُّ الْقَيُّومُ وَأَتُوبُ إِلَيْهِ
**Indonesia:** Aku memohon ampun kepada Allah, tidak ada ilah selain Dia, Yang Hidup, Yang Berdiri Sendiri, dan aku bertobat kepada-Nya.
